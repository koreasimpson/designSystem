# Web · 코리안심슨 Portfolio UI Kit

개인 포트폴리오 / 블로그 사이트의 UI 컴포넌트 키트.

## Components
- `Header.jsx` — 픽스드 상단 네비게이션 (로고 + 메뉴 + 테마 토글)
- `Hero.jsx` — 첫 화면 자기소개
- `ProjectCard.jsx` — 프로젝트 카드
- `PostListItem.jsx` — 블로그 포스트 리스트 아이템
- `Button.jsx` — 4종 버튼
- `Tag.jsx` — 태그 칩

## Screens (in `index.html`)
1. **Home** — Hero + 최근 프로젝트 + 최근 글
2. **Projects** — 프로젝트 그리드
3. **Writing** — 글 목록
4. **About** — 자기소개

상단 네비게이션 클릭으로 화면 전환 (클릭쓰루 프로토타입).
